How to run the code. Please follow below steps.

1. From the zip file submitted, copy the folder named 'code' on the desktop.
2. After we have succesfully copied the folder on desktop we have to open the folder.
3. Inside the folder open command prompt by typing 'cmd' command in the path box.
4. Run "pip install -r requirement.txt" command to install all the required packages to run the code.
5. Then run "python main.py" command to run the code.
6. After the program as started to run we will be asked to type a number,
	type 0 to run ESA,
	type 1 to run LSA.
7. All the outputs like plots, and intermideate processed file will be stored in output folder.
